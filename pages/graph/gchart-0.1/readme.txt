GChart -- an object-oriented PHP library for drawing charts using Google Chart APIs

Usage:

Unpack the zip or gz file into a directory, for example D:\gchart. In the PHP pages, include D:\gchart\GChart.php. Then you can use all the classes of GChart.

For more information, please visit the project website http://gchart.sourceforge.net and the development blog http://gchart.sourceforge.net/blog/
